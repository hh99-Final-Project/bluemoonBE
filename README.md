### 🏠 소개
- Bluemoon 서비스는 <b>익명</b> & <b>다이어리 </b> 서비스입니다.
- 익명으로 자신의 다이어리를 작성하고 고민을 서로 나눌 수 있습니다.
- 주요기능
  1. WebSocket을 이용한 실시간 채팅 및 알림
  2. Web RTC를 이용한 음성녹음 및 재생
